# NO-RISK

### Private File → JSON / Markdown Converter

A fast, browser-based file processor that converts CSV and Excel files into JSON or Markdown — **without ever uploading your data**.

---

## 🚀 What it does

NO-RISK lets you:

* Upload **multiple files (up to 15)**
* Process everything **locally in your browser**
* Convert files into:

  * JSON
  * Markdown tables
* Switch between files instantly
* Download results individually or all at once

---

## 🔒 Privacy First

* ❌ No backend
* ❌ No cloud uploads
* ❌ No localStorage usage
* ✅ Everything runs in-memory
* ✅ Data is wiped when the tab closes

Your files never leave your device.

---

## ⚡ Features

* Drag & drop or click to upload
* Supports:

  * `.csv`
  * `.xlsx`
* Multi-file support (up to 15 files, 20MB total)
* File viewer with navigation (⬅️ ➡️)
* JSON + Table view modes
* Convert all files to Markdown instantly
* Download options:

  * Current file (JSON / MD)
  * All files (JSON bundle)
* Built-in basic sensitive data detection (SSN / card patterns)

---

## 🖥️ How to Use

1. Open `index.html` in your browser
2. Drag & drop files or click the upload box
3. View data as JSON or table
4. Click **Convert to MD** to generate markdown
5. Download your results

---

## 🧠 Why NO-RISK?

Most file tools upload your data to servers.

NO-RISK is different:

> Everything happens locally — faster, safer, and private by design.
